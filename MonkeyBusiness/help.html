Moving Image


Select image

  • moving by 1 pixel step: use keys: UP arrow, Down arrow, Left arrow, Right arrow

  • moving by 10 pixels step: Shift+key

  • moving by 100 pixels step: Ctrl+key

  • set image on TOP: right mouse click


Crop Image:

Horizontally:

  • set mouse on left or right image side no more than 10pixels from the edge and double click. Follow the instructions.

Vertically:

  • set mouse on top or bottom image side no more than 10pixels from the edge and double click. Follow the instructions.


Transformations

at least one image must be selected to enable buttons.

Some transformations allows parameters change. You can use sliders or wheel if enabled.

If not, not change is possible.


Save image

allows save single image if selected


Save scene as image

saves content of the scene as you can see it.
